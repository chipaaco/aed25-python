# aed25-python

Algo bastante similar a un tetris.

## Correrlo

Solamente necesitás tener python instalado y en el path, y un intérprete de comandos (cmd, powershell, o el de tu sistema operativo) y posicionarte en la carpeta del juego.

```
python tetris.py
```
